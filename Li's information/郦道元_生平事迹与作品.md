## 二、个人生平

### 2.1 出身与早年

郦道元出生于北魏时期的官宦世家。父亲郦范为北魏立下赫赫战功，被封为"永宁侯"。优渥的家庭环境不仅提供了丰富的物质条件，更让他从小就接触到各种书籍和知识。

少年时期，郦道元跟随父亲在山东生活。他常常与朋友到有山水的地方游览，仔细观察水流走向、山川形态。每到一处，都会向当地老人请教山川河流的历史变迁、民间传说和风土人情。

### 2.2 求学与积累

郦道元勤奋好学，自幼博览群书，尤其对古代地理书籍情有独钟：

- 《山海经》——过于芜杂
- 《禹贡》、《周礼·职方》——只具轮廓
- 《汉书·地理志》——记述不详备
- 《水经》——专述河流，具系统纲领，但未记水道以外地理情况

他发现地理现象并非一成不变：河流会改道、地名会变更、城镇村庄也有兴衰。古代地理书籍已无法准确反映当时的地理状况，于是立志要撰写一部全面、准确的地理著作。

### 2.3 仕途经历

郦道元一生历任多职：

**中央官职：**
- 承袭永宁侯（后依惯例降为伯）
- 太傅掾
- 书侍御史（因御史中尉李彪举荐）
- 御史中尉
- 黄门侍郎
- 持节、兼侍中、摄行台尚书

**地方官职：**
- 冀州镇东府长史（为政严酷，奸盗逃于他境）
- 鲁阳郡守（上奏设立地方学校，崇尚教育，山上蛮人敬服其威名）
- 东荆州刺史（铁腕执政，蛮人上告其苛刻严酷，后被免官）
- 河南尹

**军事功绩：**
- 孝昌初年，梁军攻打扬州，郦道元奉命节制调度诸军，梁军攻至涡阳被击败，郦道元麾兵追击，杀敌颇多

### 2.4 为人与性格

郦道元一向以"执法清正严明"、"素有严猛之称"著称：

- **刚正不阿**：不畏权贵，敢于与不法行为作斗争
- **铁腕执政**：在地方任上以严厉著称，吏人畏惧
- **注重教育**：在鲁阳郡奏请设立学校
- **得罪权贵**：因处死汝南王元悦的男宠丘念，结下深仇

《魏书》将他列入《酷吏传》，可见其执法之严厉。

### 2.5 遇害经过

北魏孝昌三年（公元527年），汝南王元悦乘雍州刺史萧宝夤企图反叛之机，怂恿朝廷派郦道元去作关右大使（实际上是将他支开）。萧宝夤果然反叛，派人将郦道元与其弟郦道峻及两个儿子在阴盘驿亭（今陕西省临潼县东）一同杀害。

## 三、代表作品：《水经注》

### 3.1 概述

| 项目 | 内容 |
|------|------|
| **书名** | 《水经注》 |
| **性质** | 古代中国地理名著 |
| **卷数** | 原著40卷（宋初已佚5卷，今本仍作40卷） |
| **字数** | 30余万字 |
| **成书背景** | 以汉代《水经》为纲，加以详细注释和扩充 |
| **参考资料** | 参阅437种书籍，辑录汉魏金石碑刻350种左右 |

### 3.2 内容规模

- **河流**：1252条（原《水经》仅137条，增加了近千条）
- **湖泊/沼泽**：500余处
- **泉水/井水**：近300处
- **伏流**：30余处
- **瀑布**：60多处
- **山岳/丘阜地名**：近2000处
- **喀斯特洞穴**：70余处
- **植物品种**：140余种
- **动物种类**：超过100种
- **县级城市和城邑**：2800座
- **古都**：180座
- **聚落**（镇/乡/亭/里/聚/村等）：约1000处
- **桥梁**：约100座
- **津渡**：近100处
- **战役记载**：不下300次
- **水灾记录**：30多次
- **地震记录**：近20次
- **古塔**：30多处
- **宫殿**：120余处
- **陵墓**：260余处
- **寺院**：26处

### 3.3 涵盖领域

**自然地理：**
- 河流发源到入海的完整信息：干流、支流、河谷宽度、河床深度、水量、水位季节变化、含沙量、冰期
- 各种地貌类型：山、岳、峰、岭、坂、冈、丘、阜、原、川、野、平原等
- 自然灾害：水灾、旱灾、风灾、蝗灾、地震

**人文地理：**
- 政区建置（补充正史地理志之不足）
- 国外城市（如印度的波罗奈城、王舍新城等）
- 水运和陆路交通
- 农田水利工程（坡湖、堤、塘、堰等）
- 手工业生产（采矿、冶金、纺织、造币、食品等）
- 矿物资源（金属矿、非金属矿、煤炭、石油、天然气等）
- 兵要地理（古今大小战役300余次）

**学科资料：**
- 地名约2万处（解释地名2400多处）
- 碑刻墨迹、渔歌民谣
- 民间歌谣、谚语方言、传说故事

### 3.4 编写特点

1. **例义谨严**：经文以"过"字叙述水流行经之地，注文以"迳"字叙述，全书无一苟且
2. **实践第一**：确定"迳见"（目验）的鉴定标准，强调亲历所见的科学态度
3. **大胆质疑**：提出"经之误正"论点，敢于指出经文水道的讹误
4. **金石之学**：开创碑刻文物考证，成为宋代金石学的先导

### 3.5 成书动机

- **学术追求**：古代地理书籍各有所短，无法满足对地理知识的渴望
- **与时俱进**：地理现象不断变化，需以新著反映当代情况
- **统一理想**：身处分裂时代，利用河流水系这一自然因素打破政治疆界限制，体现祖国统一的决心

### 3.6 文学价值

《水经注》不仅是科学名著，也是文学艺术珍品：

- **"写水着眼于动态，写山则致力于静态"**
- 是魏晋南北朝时期山水散文的集锦、神话传说的荟萃、名胜古迹的导游图、风土民情的采访录
- 语言运用出类拔萃，仅描写瀑布就用：泷、洪、悬流、悬水、悬涛、悬泉、悬涧、悬波、颓波、飞清等词汇
- 文笔绚烂，语言清丽，"词组只字，妙绝古今"

### 3.7 历代评价

- **苏轼**："嗟我乐何深，水经也屡读。"
- **张岱**："古人记山水，太上有郦道元，其次柳子厚，近时则袁中郎。"
- **刘熙载**："郦道元叙山水，峻洁层深，奄有楚辞《山鬼》《招隐士》胜境。柳州游记，此先导也。"
- **日本地理学家米仓二郎**："中世纪全世界最伟大的地理学家。"
- **李白、杜甫**的诗均吸收了《水经注》的艺术滋养
- **柳宗元**的《永州八记》实脱胎于《水经注》

### 3.8 版本流传

**钞本：**
- 隋朝《隋书·经籍志》著录40卷足本（最早钞本）
- 唐朝为国家藏书
- 北宋景祐年间崇文院整理为35卷（开始残缺）
- 明代《永乐大典》钞本（现存最早钞本）
- 清代"小山堂五校钞本"（全祖望33岁时作品，熔全祖望与赵一清业绩于一炉）

**刻本：**
- 明代嘉靖十三年黄省曾刊本
- 明代万历四十三年朱谋玮《水经注笺》（名本）
- 清代乾隆十九年赵一清《水经注释》
- 清代武英殿聚珍本（殿本）
- 清代全祖望《七校本水经注》（光绪十四年刊行）
- 清代王先谦《合校本水经注》（光绪十八年，集大成之作）

### 3.9 后世研究（郦学）

明清以来，研究《水经注》形成专门的"郦学"：

- **戴震**：删妄增之字一千多个，改正错讹三千多处，补葺阙佚两千多处
- **杨守敬、熊会贞**：毕生撰写《水经注疏》，编绘《水经注图》
- **陈桥驿**：现代郦学大家，复校《水经注疏》

## 五、历史地位与影响

1. **地理学**：中国古代最全面、最系统的综合性地理著作，可研究水道变迁、湖泊湮废、地下水开发、海岸变迁、城市规划、历史时期气候变化等
2. **文学**：山水散文的开创者，影响了李白、杜甫、柳宗元、苏轼等文学巨匠
3. **历史学**：记载大量正史未载的内容，可补充正史地理志之不足
4. **考古学**：记载了大量古塔、宫殿、陵墓、寺院，有考古价值
5. **地名学**：记录约2万处地名，解释地名2400多处
6. **水利学**：记载大量农田水利工程，对研究古代水利建设有重要价值
7. **金石学**：开创碑刻文物考证之先河
8. **军事学**：记载300余次战役，体现利用地形的重要性
